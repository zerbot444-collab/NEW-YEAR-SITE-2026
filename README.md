## 📌 About The Project
A modern, beautiful, and fully responsive web application built to easily view the **2026 Sinhala and Tamil New Year Auspicious Times (අලුත් අවුරුදු නැකැත්)**. 

🎉 **Huge Milestone:** Thank you for the amazing support! We recently hit a milestone of **457 users** with a **100% usage rate**! ❤️

## ✨ Key Features
* **🌍 Bilingual Support:** Seamlessly switch between Sinhala and Tamil languages.
* **⏳ Live Countdown Timer:** Real-time countdown for upcoming auspicious times (Nakath).
* **✅ Status Tracking:** Automatically marks passed auspicious times with a "Passed" badge and distinct UI changes.
* **🎵 Festive Audio:** Built-in traditional New Year background music (Aurudu Remix) with a play/mute toggle button.
* **🍂 Custom Animations:** Beautiful floating sweets (Kavum, Kokis, Banana) in the background to bring the festive vibe alive.
* **📱 Fully Responsive:** Works flawlessly on both mobile devices and desktop screens.
