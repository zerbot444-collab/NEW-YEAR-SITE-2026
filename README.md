## 📌 About The Project
> A modern, beautiful, and fully responsive web application built to easily view the **2026 Sinhala and Tamil New Year Auspicious Times (අලුත් අවුරුදු නැකැත්)**. 

🎉 **Huge Milestone:** Thank you for the amazing support! We are proudly tracking **live views** above and maintaining a **100% usage rate**! ❤️

---

## ✨ Key Features

| Feature | Description |
| :--- | :--- |
| 🌍 **Bilingual Support** | Seamlessly switch between Sinhala and Tamil languages. |
| ⏳ **Live Countdown** | Real-time animated countdown for upcoming auspicious times. |
| ✅ **Status Tracking** | Automatically marks passed events with a "Passed" badge. |
| 🎵 **Festive Audio** | Built-in traditional background music with a custom toggle UI. |
| 🍂 **Custom Animations** | Floating sweets (Kavum, Kokis, Banana) for a festive vibe. |
| 📱 **Responsive UI** | Flawless Glassmorphism design for both mobile & desktop. |
